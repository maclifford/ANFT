Convention: lowercase, hyphenated first-last name, .webp extension
(e.g. jane-doe.webp). This matches the src paths in who-we-are.html
and our-trainers.html.

Square-ish portraits work best; cards display them at 96x96, so anything
up to ~400x400 is plenty (current photos are ~144-250px). Keep each file
lightweight (well under ~100KB). Cards fall back to a monogram until a
matching photo file exists.
